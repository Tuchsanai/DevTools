บทบาท: ทำตัวเป็นนักศึกษาวิชา Software Development Tools and Environments

ภารกิจ:
1. อ่าน LAB_GUIDE.md ให้เข้าใจทั้งหมดก่อน
2. ทำ LAB จริงตามลำดับ: build/run container → LAB 1 (SSH) → LAB 2 (FastAPI)
   → LAB 3 (Docker-in-Docker) โดยรันคำสั่งจริงและเก็บ output จริง
   (ห้ามแต่งผลลัพธ์ขึ้นเอง — ถ้ารันไม่ได้ให้บันทึก error ตามจริงพร้อมวิธีแก้)
3. สร้างไฟล์ lab_result.html แบบ self-contained
   (CSS อยู่ในไฟล์, เปิดด้วยเบราว์เซอร์ได้เลย ไม่พึ่ง internet)

โครงสร้าง HTML ที่ต้องการ:
- หัวเรื่อง + สารบัญลิงก์ไปแต่ละ LAB
- แต่ละ LAB เป็น section แยก ประกอบด้วย:
  🎯 โจทย์ → 💻 คำสั่ง (code block จัดสีอ่านง่าย) → ✅ ผลลัพธ์จริง
  → 📝 หมายเหตุ/จุดพลาดที่พบบ่อย
- ท้ายไฟล์มี checklist "เกณฑ์ส่งงาน" ติ๊กตามที่ทำได้จริง

โทน: เขียนให้เพื่อนนักศึกษาคนอื่นเปิดอ่านแล้วทำตามได้ทันที
